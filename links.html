I enjoy finding nice domain names that are not occupied yet. For the moment, I have post-quantum-crypto.com which I mainly use as a hub for subdomains:

  • krijn.post-quantum-crypto.com contains a very minimal description of my papers (as does jonathan.post-quantum-crypto.com for his)
  • nist.post-quantum-crypto.com contains all schemes currently in the NIST competition for digital signatures, and each scheme has its own hyperlink (i.e. meds.post-quantum-crypto.com links to the official MEDS page)
  • specific presentations of papers, such as MEDS, get a hyperlink directly through the HTML5 presentation. See for example siam.post-quantum-crypto.com. In general, I highly recommend using Keynote + HTML5 output to design and present talks.

Other domains are

  • isogeni.es, which hosts a list of most recent eprints on isogeny-based crypto
  • isogeny.club, which hosts the webpage for The Isogeny Club which I ran with Jonathan until 2024, when I handed over my part to Maria.
